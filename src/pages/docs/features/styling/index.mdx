# Styling

material-table has a few styling options to apply styles to some components. 

| Field                       | Type          | Description                             |  
|:----------------------------|:--------------|:----------------------------------------|
| column.cellStyle            | style of func | style to apply specified cells          |
| column.headerStyle          | style         | style to apply specified column header  |
| options.headerStyle         | style         | style to apply all header cells         |
| options.rowStyle            | style of func | style to apply all rows cells           |
| options.searchFieldStyle    | style         | style to apply search textfield         |
