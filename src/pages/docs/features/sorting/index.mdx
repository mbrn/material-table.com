# Sorting

material-table has sorting feature that lets users sort data by any column.

## Usage

To make sorting available you must open sorting in options.

```jsx
import MaterialTable from 'material-table';

<MaterialTable
    // other props
    options={{
      sorting: true
    }}
/>
```

> Sorting feature is open by default

<br />

## Hints 

* You can add a default sort value to a column by setting `column.defaultSort` with `asc` or `desc`
* You can close sorting for any column by setting `column.sorting` as `false`
* You can change sorting algorithm by set a function to `column.customSort`
