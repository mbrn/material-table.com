# Export

material-table has built-in export feature. It supports pdf and csv formats only. 

## Usage

To make export available you must open exportButton

```jsx
import MaterialTable from 'material-table';

<MaterialTable
    // other props
    options={{
      exportButton: true
    }}
/>
```

<br />

## Hints 

* You can change export file name by setting `options.exportFileName`
* You can change csv delimeter by setting `options.exportDelimiter`
