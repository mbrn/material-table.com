# Detail Panel

You can add a detail panel to every row by using detailPanel feature.

## Usages

<br/>

### One detail panel as a functions

If you neeed just one type of detail panel, you can just set a function that returns a React element. 

```jsx
<MaterialTable
  // other props
  detailPanel={rowData => {
    return (
      <iframe
        width="100%"
        height="315"
        src="https://www.youtube.com/embed/C0DPdy98e4c"
        frameborder="0"
        allow="accelerometer; autoplay; encrypted-media; gyroscope; picture-in-picture"
        allowfullscreen
      />
    )
  }}
/>
```

<br/>

### Multiple detail panel as array of detail panels

If you neeed more than one type of detail panel, you can set array of detailPanel objects. 

```jsx
<MaterialTable
  // other props
  detailPanel={[
    {
      tooltip: 'Show Name',
      render: rowData => {
        return (
          <div
            style={{
              fontSize: 100,
              textAlign: 'center',
              color: 'white',
              backgroundColor: '#43A047',
            }}
          >
            {rowData.name}
          </div>
        )
      },
    },
    {
      icon: 'account_circle',
      tooltip: 'Show Surname',
      render: rowData => {
        return (
          <div
            style={{
              fontSize: 100,
              textAlign: 'center',
              color: 'white',
              backgroundColor: '#E53935',
            }}
          >
            {rowData.surname}
          </div>
        )
      },
    },
    {
      icon: 'favorite_border',
      openIcon: 'favorite',
      tooltip: 'Show Both',
      render: rowData => {
        return (
          <div
            style={{
              fontSize: 100,
              textAlign: 'center',
              color: 'white',
              backgroundColor: '#FDD835',
            }}
          >
            {rowData.name} {rowData.surname}
          </div>
        )
      },
    },
  ]}
/>
```
