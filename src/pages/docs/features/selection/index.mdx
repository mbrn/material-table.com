# Selection

material-table has selection feature that lets users to select multiple rows.

## Usage

To make selection available you must open selection in options.

```jsx
import MaterialTable from 'material-table';

<MaterialTable
    // other props
    options={{
      selection: true
    }}
/>
```

<br />

## Hints 

* You can use `onSelectionChange` to handle selection changes
